# 线程

    Linux 实现线程的机制非常独特。从内核的角度来说，它并没有线程这个概念。Linux 把所有的线程都当做进程来实现。
内核并没有准备特别的调度算法或是定义特别的数据结构来表征线程。

    线程仅仅被视为一个与其他进程共享某些资源的进程。每个线程都拥有唯一隶属于自己的 ```task_struct```，在内核中，它
  看起来就像是一个普通的进程。
